## 期待値で宝くじを評価する（5–17分）

### 期待値とは何か

学年によっては期待値を未習の場合があるため，言葉と例から入る．

```{tip} 定義：期待値【板書】
ある賭けでもらえる金額の「平均」を<span style="color:red">期待値</span>といい，

$$
\text{期待値} = (\text{金額} \times \text{その確率}) \text{ をすべての場合について足したもの}
$$

で計算する．「たくさん繰り返したとき，1回あたり平均いくらになるか」を表す量である．
```

```{note} 担当者向け：記号で書くなら
結果 $x_1, \dots, x_n$ が確率 $p_1, \dots, p_n$ で起こるとき $E = p_1 x_1 + p_2 x_2 + \dots + p_n x_n$．
Σ記号は使わない（数学B未習者がいる前提）．「大学ではこれを $\sum$ という記号で1行に書く」と一言添えると背伸び感が出る．
```

◆ ウォームアップ（2分）：サイコロの出た目 $\times 100$ 円もらえるとする．期待値は

$$
\frac{1}{6}(100+200+300+400+500+600) = \frac{2100}{6} = 350 \text{ 円}
$$

▶ 参加費が400円なら，このゲームをやるか？　平均350円しか戻らないので，長い目で見れば損である．
ここで「**期待値と参加費を比べる**」という判断の型を板書に残す．

**宝くじを全部買ったら，平均してこれだけ損する**


### 宝くじの期待値を計算してみる

実際の宝くじは等級が多く複雑なので，年末ジャンボ宝くじを簡易化したもので考える．

```{tip} 例：年末ジャンボ宝くじ（一部）【板書：この表は残す】
1枚 **300円**．当選確率と賞金を次のように定める．

| 等級 | 賞金 | 当選確率 |
| --- | --- | --- |
| 1等 | 
| 2等 | 
| 3等 | 
| はずれ | 0円 | 残り |
```

◆ もらえる金額の期待値を**受講者に計算させる**（賞金 $\times$ 確率 を3つ足すだけ）．

$$
\begin{aligned}
E[\text{賞金}]
&= 1\text{億} \times \frac{1}{1000\text{万}} + 100\text{万} \times \frac{1}{10\text{万}} + 3000 \times \frac{1}{100} + 0 \\
&= 10 + 10 + 30 = 50 \text{ 円}
\end{aligned}
$$

1枚300円払って，平均50円しか戻ってこない．

```{note} 還元率という見方（口頭中心，数字だけ板書）
$$
\text{還元率} = \frac{\text{もらえる金額の期待値}}{\text{購入金額}} = \frac{50}{300} \approx 17\%
$$

このミニ宝くじは極端に単純化してあるので17%だが，実際の日本の宝くじも当選金の合計は法律で売上の5割未満と決められており，還元率は<span style="color:red">約45〜47%</span>（残りは公共事業・経費等にあてられる）．
```

### 期待値だけで選ぶ人【板書】

期待値が高いほうを選ぶ，という基準で判断する人を**リスク中立的**（risk neutral）という．この基準では

$$
\text{期待値}(50\text{円}) < \text{価格}(300\text{円})
$$

なので，**宝くじは買うべきでない**という結論になる．

▶ ところが現実には大勢が買う．**この基準のどこが足りないのか**．ここから大学の話に入る．

## 解決のカギ：「お金の価値」は金額に比例しない（17–28分）

### 100万円の重みは状況で変わる

▶ 思考実験：所持金**0円のあなた**にとっての100万円と，**1億円持っているあなた**にとっての100万円は，同じ「うれしさ」だろうか．

多くの人は「最初の100万円のほうがずっとありがたい」と答える．
つまり「うれしさ」は金額に比例しない．ならば，**金額ではなく「うれしさ」で期待値を取り直せばよい**．

```{tip} 定義：効用関数と期待効用【板書】
金額 $x$ を手にしたときの満足度・うれしさを数値化したものを<span style="color:red">効用</span>といい，その対応 $u(x)$ を<span style="color:red">効用関数</span>という．

意思決定は「金額の期待値」ではなく，**期待効用**

$$
\text{期待効用} = (\text{効用 } u(x) \times \text{その確率}) \text{ をすべて足したもの}
$$

を最大化する，と考えると人間の行動をうまく説明できる．18世紀の数学者ベルヌーイの発想であり，のちにフォン・ノイマンとモルゲンシュテルンが数学的に定式化した**期待効用理論**の出発点である．
```

### 効用関数の「形」がすべてを語る

お金が増えるほど効用も増えるが，**増え方がだんだん鈍る**（最初の100万円ほどありがたい）．
これは効用関数のグラフが**上にふくらんだカーブ**（上に凸）であることに対応する．

```{tip} 例：$u(x)=\sqrt{x}$ で「鈍り」を見る【板書：表とグラフ】
$x$ の単位を万円として

| $x$（万円） | 0 | 1 | 4 | 9 | 100 |
| --- | --- | --- | --- | --- | --- |
| $u(x)=\sqrt{x}$ | 0 | 1 | 2 | 3 | 10 |

- お金が $1\to 4$（$+3$）で効用は $1\to 2$（$+1$）．$4\to 9$（$+5$）でも効用は $+1$．
- **同じ「効用 $+1$」を得るのに，より多くのお金が必要になっていく** ＝ 満足の伸びが鈍る．
```

板書：右肩上がりだが上にふくらんだカーブを描く（表の5点を打ってから結ぶと速い）．

```{figure} figures/utility_sqrt.png
:alt: u(x)=sqrt(x) の効用関数
:width: 90%
```

### リスクへの態度＝カーブの曲がり方【板書：3つの図を並べる】

- <span style="color:red">リスク回避的</span>：グラフが**上にふくらむ**．確実を好む．
- <span style="color:red">リスク中立的</span>：グラフが**直線**．期待値で判断する．
- <span style="color:red">リスク愛好的</span>：グラフが**下にふくらむ**．賭けを好む．

```{note} 担当者向け
数学IIIの言葉では順に $u''<0,\ u''=0,\ u''>0$ だが，授業では使わない．「曲がり方」という言葉とグラフの形だけで通す．
```

◆ 確かめる賭け：**「確実に50万円」** と **「50%で100万円，50%で0円」**（どちらも期待値50万円）．挙手でどちらを選ぶか聞いてから，$u(x)=\sqrt{x}$（単位：万円）で期待効用を比べる．

$$
u(50) = \sqrt{50} = 5\sqrt{2} \approx 7.1, \qquad
\text{くじの期待効用} = \tfrac{1}{2}\sqrt{100} + \tfrac{1}{2}\sqrt{0} = 5
$$

$7.1 > 5$ なので，この効用関数をもつ人は**確実な50万円を選ぶ＝リスク回避的**である．挙手でも多数派はこちらのはず．
（$\sqrt{2} \approx 1.41$ は「ひとよひとよに」で暗算できる．電卓不要）

```{note} 保険の話（口頭1分）
リスク回避的な人は，期待値より高い保険料を払ってでも不確実性を消したいと考える．**保険という巨大産業は，このカーブの上へのふくらみの上に成り立っている**．
ここで一言だけ謎を深めておく：「リスク回避的なら保険は買っても宝くじは買わないはず．なのに同じ人が両方買う．これも実は説明できる（→発展）」
```

## 体験：自分のリスク態度を測ってみよう（28–42分，このコマの山場）

▶ 「人によって効用関数は違う」と言ってきた．**では，あなたの効用関数は**．実際に測ってみよう．
これは行動経済学で実際に使われる**確実性等価法**の簡易版である．紙と鉛筆だけでできる．

### 基準を決める【板書】

$u(0\text{円}) = 0$，$u(100\text{万円}) = 1$ とする（効用の「ものさし」は自分で決めてよい）．

### 質問は1つだけ

◆ **質問（中点を探す）**：各自，紙に金額を1つ書く．

> 「50%で100万円・50%で0円」のくじと，「確実にもらえる $X$ 円」のどちらかを選べる．$X$ がいくらなら**どちらでもよい**（迷う）と感じるか．

その金額を $c$ とする（くじと同じ価値に感じる確実な金額なので，<span style="color:red">確実性等価</span>と呼ぶ）．定義より

$$
u(c) = \tfrac{1}{2}\times u(100\text{万}) + \tfrac{1}{2}\times u(0) = \tfrac{1}{2}\times 1 + \tfrac{1}{2}\times 0 = 0.5
$$

$\to$ 「$c$ 円の効用は 0.5」という，あなたの効用関数上の1点が決まる．

### 3点で「自分のタイプ」が分かる【板書：判定図】

$(0, 0)$，$(c, 0.5)$，$(100\text{万}, 1)$ の3点を，横軸＝金額・縦軸＝効用で打つ．

```{important} 読み取り方【板書】
- $c$ が **50万円より小さい**（例：30万円）$\to$ 3点は**上にふくらむ** $\to$ <span style="color:red">リスク回避的</span>
- $c$ が **ほぼ50万円** $\to$ ほぼ**直線** $\to$ <span style="color:red">リスク中立的</span>
- $c$ が **50万円より大きい** $\to$ **下にふくらむ** $\to$ <span style="color:red">リスク愛好的</span>
```

```{important} 今日の謎への答え【板書：冒頭の謎の横に書く】
1. 判断の基準は金額の期待値ではなく**期待効用**（うれしさの期待値）．
```

## 付録B：発展 — 保険も宝くじも買う人の謎（時間が余った場合・質問対応用）

1948年に経済学者フリードマンとサヴェッジが指摘した有名な謎：**リスク回避的な人なら，保険は買っても宝くじ（不利な賭け）は買わないはず**．なのに**同じ人が保険にも宝くじにも払う**．

```{important} フリードマン＝サヴェッジの効用関数
効用関数が**全体としては上にふくらむが，途中に下にふくらむ区間がある** —— 曲がり方が途中で反転する**ねじれ**をもつと考えると説明できる．

- いまの生活水準のまわり（小さな損得）では**上にふくらむ＝リスク回避** $\to$ 保険を買う．
- 「一発当てれば人生が変わる」大金の領域では**下にふくらむ＝リスク愛好** $\to$ 宝くじを買う．
```

```{figure} figures/utility_friedman_savage.png
:alt: フリードマン＝サヴェッジ型の効用関数
```

## 付録C：担当者用 — 大学ではこう発展する（Python・授業では使わない）

板書授業のため受講者には見せないが，まとめで「情報数理学科ではこれをプログラミングでやる」と口頭紹介する際の種，および本編講義との接続用に残す．

:::{seealso} モンテカルロで「平均は50円」を体感する
理論上の期待値50円を，実際に何百万回もくじを引いて平均を取ることで確かめられる（大数の法則）．

```python
import numpy as np

rng = np.random.default_rng(0)
N = 10_000_000          # 1000万回くじを引く

u = rng.random(N)       # 0〜1の一様乱数
prize = np.where(u < 1/10_000_000, 1e8,                          # 1等 1億円
         np.where(u < 1/10_000_000 + 1/100_000, 1e6,             # 2等 100万円
         np.where(u < 1/10_000_000 + 1/100_000 + 1/100, 3000, 0)))  # 3等

print("1枚あたり平均賞金:", prize.mean(), "円")   # だいたい 50 円に近づく
print("購入額300円との差:", prize.mean() - 300, "円")
```

「平均は50円でも，ほとんどの人は0円，ごく一部が大当たり」という分布の偏りも見せられる．
:::

:::{seealso} 確実性等価の5点に効用関数をフィッティングする
体験コーナーで集めた点に，べき乗の効用関数 $u(x) = (x/100)^{\alpha}$（$x$ は万円）を当てはめると，たった1つの数 $\alpha$（リスク態度パラメタ）で効用関数を要約できる．

```python
import numpy as np
from scipy.optimize import curve_fit

# 例：ある受講者が答えた確実性等価（万円）。u=0.25,0.5,0.75 に対応
money = np.array([0, 30, 50, 72, 100])   # 自分の確実性等価に置き換える
util  = np.array([0, 0.25, 0.5, 0.75, 1.0])

def u(x, alpha):
    return (x / 100.0) ** alpha

alpha, _ = curve_fit(u, money, util, p0=[1.0], bounds=(0.05, 5))
print(f"あなたの alpha = {alpha[0]:.2f}")
print("alpha<1: リスク回避 / =1: 中立 / >1: リスク愛好")
```

クラス全員の $\alpha$ を集めてヒストグラムにすれば「教室の意思決定データ」ができる．大学の[第2回](/contents/2_expected_utility)では，これをさらに本格化した「CRRA効用関数による最尤推定」を扱う．
:::

## 付録D：想定問答（Q&A）

```{note} Q. 期待値が損でも，当たる人は実際にいますよね？
A. その通り．期待値は「全員を平均した」値で，個人の結果は当たり外れに大きくばらつく．「ほとんどの人が0円，ごく少数が巨額」という偏った分布になる．期待値だけでなく**分布の形**を見るのが大事，というのも今日の裏テーマである．
```

```{note} Q. では宝くじは買わないほうがいい？
A. 数学は「正解」を押し付けない．**自分の効用関数と，夢という体験をどう評価するか**で答えは変わる．「無理のない範囲で夢を買う」のは，その人にとって合理的でありうる．大事なのは**自分の判断を説明できること**である．（教育的配慮：賭け事を勧める授業ではないことを明言し，「期待値的に不利と知った上で判断する力」が主旨だと締める）
```

```{note} Q. 効用関数は本当に測れるの？
A. 今日の確実性等価法は実際の研究手法の簡易版である．より精密には多数の質問・実験で推定する．測り方そのものが研究テーマであり，心理学・経済学・統計学が交わる面白い領域である．
```

```{note} Q. 期待値を授業で習っていないのですが
A. 今日の定義（金額×確率を足す）だけで十分．高校の数学A・Bでも登場する考え方なので，先取りだと思ってほしい．
```
