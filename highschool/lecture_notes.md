## まず「期待値」で宝くじを評価する

### 期待値の復習

```{tip} 定義：期待値
ある賭けで，結果 $x_1, x_2, \dots, x_n$ がそれぞれ確率 $p_1, p_2, \dots, p_n$ で起こるとき，もらえる金額の平均を<span style="color:red">期待値</span>といい，

$$
E[X] = \sum_{k=1}^{n} p_k x_k = p_1 x_1 + p_2 x_2 + \dots + p_n x_n
$$

と定める．「たくさん繰り返したとき，1回あたり平均いくらになるか」を表す量である．
```

◆ ウォームアップ：サイコロの出た目 $\times 100$ 円もらえるとする．期待値は

$$
E[X] = \frac{1}{6}(100+200+300+400+500+600) = \frac{2100}{6} = 350 \text{ 円}
$$

▶ 参加費が400円なら，このゲームをやるか？　平均350円しか戻らないので，長い目で見れば損である．

### 宝くじの期待値を計算してみる

実際の宝くじは等級が多く複雑なので，まず本質だけを取り出した**ミニ宝くじ**で考える．

```{tip} 例：ミニ宝くじ
1枚 **300円**．当選確率と賞金を次のように定める．

| 等級 | 賞金 | 当選確率 |
| --- | --- | --- |
| 1等 | 1億円 | $1/10{,}000{,}000$ |
| 2等 | 100万円 | $1/100{,}000$ |
| 3等 | 3,000円 | $1/100$ |
| はずれ | 0円 | 残り |
```

もらえる金額の期待値は，賞金 $\times$ 確率 の和として

$$
\begin{aligned}
E[\text{賞金}]
&= 10^8 \times \frac{1}{10^7} + 10^6 \times \frac{1}{10^5} + 3000 \times \frac{1}{100} + 0 \\
&= 10 + 10 + 30 = 50 \text{ 円}
\end{aligned}
$$

となる．1枚300円払って，平均50円しか戻ってこない．

```{note} 還元率という見方
$$
\text{還元率} = \frac{\text{もらえる金額の期待値}}{\text{購入金額}} = \frac{50}{300} \approx 0.17
$$

このモデルでは「払った額の約17%しか平均では戻らない」．
実際の日本の宝くじの還元率は<span style="color:red">約45〜47%</span>（残りは公共事業・経費等にあてられる）．参考までに公営競技は約75%前後で，宝くじは賭けの中でも特に期待値的に不利な部類に入る．
```

### 「期待値だけで選ぶ人」の結論

```{tip} 定義：リスク中立的な意思決定
「期待値が高いほうを選ぶ」という基準をもつ意思決定者を<span style="color:red">リスク中立的</span>（risk neutral）という．この基準では

$$
\text{期待値}(50\text{円}) < \text{価格}(300\text{円})
$$

なので，**宝くじは買うべきでない**という結論になる．
```

▶ ところが現実には大勢が買う．**この基準のどこが足りないのか**．ここから大学の話に入る．

```{note} 板書・電卓だけで示す数値例（PC不要）
電卓で「賞金 $\times$ 確率」を3つ足すだけ：$10 + 10 + 30 = 50$．確率の桁（1千万分の1 など）を受講者に声に出させると，「当たらなさ」が体感できる．
```

:::{seealso} （PCがあれば）モンテカルロで「平均は50円」を体感する
理論上の期待値50円を，実際に何百万回もくじを引いて平均を取ることで確かめられる．これがモンテカルロ・シミュレーションである．

```python
import numpy as np

rng = np.random.default_rng(0)
N = 10_000_000          # 1000万回くじを引く

u = rng.random(N)       # 0〜1の一様乱数
prize = np.where(u < 1/10_000_000, 1e8,                          # 1等 1億円
         np.where(u < 1/10_000_000 + 1/100_000, 1e6,             # 2等 100万円
         np.where(u < 1/10_000_000 + 1/100_000 + 1/100, 3000, 0)))  # 3等

print("1枚あたり平均賞金:", prize.mean(), "円")   # だいたい 50 円に近づく
print("購入額300円との差:", prize.mean() - 300, "円")
```

実行のたびに値は少し揺れるが，回数を増やすほど理論値50円に近づく（大数の法則）．「平均は50円でも，ほとんどの人は0円，ごく一部が大当たり」という分布の偏りも見せられる．
:::

## 解決のカギ：「お金の価値」は金額に比例しない

### 100万円の重みは状況で変わる

▶ 思考実験：手元のお金が**0円のあなた**にとっての100万円と，**1億円持っているあなた**にとっての100万円は，同じ「うれしさ」だろうか．

多くの人は「最初の100万円のほうがずっとありがたい」と答える．

```{tip} 定義：効用関数
金額そのものの額 $x$ ではなく，それを手にしたときの満足度・うれしさを数値化したものを<span style="color:red">効用</span>といい，$x$ に効用を対応させる関数 $u(x)$ を<span style="color:red">効用関数</span>という．

意思決定は「金額の期待値」ではなく，**期待効用**

$$
E_u(L) = \sum_{k=1}^{n} p_k\, u(x_k)
$$

を最大化する，と考えると人間の行動をうまく説明できる．これが18世紀の数学者ベルヌーイの発想であり，のちにフォン・ノイマンとモルゲンシュテルンが数学的に定式化した**期待効用理論**の出発点である．
```

```{note} なぜベルヌーイはこれを考えたか
「コインを表が出るまで投げ，$n$ 回目で初めて表なら $2^n$ 円」というゲームは，期待値が $\sum_{n=1}^{\infty} \frac{1}{2^n}\cdot 2^n = \infty$ と発散するのに，大金を払う人はいない（**サンクトペテルブルクのパラドックス**）．「金額そのものではなく満足度で測る」と考えると，この矛盾も解ける．詳しくは大学の[期待効用理論](/contents/2_expected_utility)で扱う．
```

### 効用関数の「形」がすべてを語る

お金が増えるほど効用も増えるが，**増え方がだんだん鈍る**．これを**限界効用逓減**といい，効用関数が**上に凸**（凹関数）であることに対応する．

```{tip} 例：$u(x)=\sqrt{x}$ で「鈍り」を見る
- $u(0)=0,\; u(1)=1,\; u(4)=2,\; u(9)=3,\; u(100)=10$
- お金が $1\to 4$（$+3$）で効用は $1\to 2$（$+1$）．$4\to 9$（$+5$）でも効用は $+1$．
- **同じ「効用 $+1$」を得るのに，より多くのお金が必要になっていく** ＝ 満足の伸びが鈍る．
```

### リスクへの態度＝カーブの曲がり方

```{tip} 定義：リスクに対する態度
効用関数の形状により，意思決定者のリスクへの態度を分類できる．

- <span style="color:red">リスク回避的</span>：効用関数が上に凸（凹関数）．確実を好む．$u''(x)<0$
- <span style="color:red">リスク中立的</span>：効用関数が直線．期待値で判断する．$u''(x)=0$
- <span style="color:red">リスク愛好的</span>：効用関数が下に凸（凸関数）．賭けを好む．$u''(x)>0$
```

◆ 確かめる賭け：**「確実に50万円」** と **「50%で100万円，50%で0円」**（どちらも期待値50万円）．
$u(x)=\sqrt{x}$（単位：万円）で期待効用を比べる．

$$
u(50) = \sqrt{50} \approx 7.07, \qquad
E_u(\text{くじ}) = \tfrac{1}{2}\sqrt{100} + \tfrac{1}{2}\sqrt{0} = \tfrac{1}{2}\times 10 = 5.0
$$

$7.07 > 5.0$ なので，この人は**確実な50万円を選ぶ＝リスク回避的**である．多くの人がこちらを選ぶ．

```{note} 確実性等価とリスクプレミアム（少し背伸び）
くじと同じ効用を与える「確実な金額」を<span style="color:red">確実性等価</span> $c$ という：$u(c)=E_u(\text{くじ})$．上の例では $\sqrt{c}=5.0 \Rightarrow c = 25$ 万円．つまりこの人は，期待値50万円のくじを**たった25万円の確実なお金**と同じ価値にしか見ていない．差 $50-25=25$ 万円が，リスクを引き受ける対価＝<span style="color:red">リスクプレミアム</span>である．

**保険**はこの裏返しで，人は期待値より高い保険料を払ってでも不確実性を消したいと考える．リスク回避は，保険が成り立つ理由そのものである．
```

▶ ここで矛盾が見えてくる：**リスク回避的な人なら，保険は買っても，宝くじ（不利な賭け）は買わないはず**．なのに**同じ人が保険にも宝くじにも払う**．なぜか．

## 奥行き：保険も宝くじも買う人の謎

これは1948年に経済学者フリードマンとサヴェッジが指摘した有名な謎である．ノーベル賞級の問いが，宝くじ売り場の前にころがっている．

```{important} フリードマン＝サヴェッジの効用関数
「同じ人が，保険（リスク回避）も宝くじ（リスク愛好）も買う」事実は，効用関数が**全体としては上に凸だが，途中に下に凸の区間がある** —— つまり途中で曲がり方が反転する**ねじれ**をもつと考えると説明できる．

- いまの生活水準のまわり（小さな損得）では**上に凸＝リスク回避** $\to$ 保険を買う．
- 「一発当てれば人生が変わる」大金の領域では**下に凸＝リスク愛好** $\to$ 宝くじを買う．
```

## 体験：自分の効用関数を推定してみよう（このコマの山場）

▶ 「人によって効用関数は違う」と言ってきた．**では，あなたの効用関数は**．実際に測ってみよう．
これは行動経済学で実際に使われる**確実性等価法**の簡易版である．

### やり方（紙と鉛筆だけ）

基準を決める：$u(0\text{円}) = 0$，$u(100\text{万円}) = 1$ とする（効用の「ものさし」は自分で決めてよい）．

◆ **質問1（中点を探す）**

> 「50%で100万円・50%で0円」のくじと，「確実にもらえる $X$ 円」のどちらかを選べる．$X$ がいくらなら**どちらでもよい（迷う）**と感じるか．

その金額を $c_1$ とすると，定義より

$$
u(c_1) = \tfrac{1}{2}\,u(100\text{万}) + \tfrac{1}{2}\,u(0) = \tfrac{1}{2}\times 1 + \tfrac{1}{2}\times 0 = 0.5
$$

$\to$ 「$c_1$ 円の効用は 0.5」という，あなたの効用関数上の1点が決まる．

◆ **質問2・3（点を増やす）**

> - 「50%で $c_1$ 円・50%で0円」で迷う確実額 $c_2$ $\to$ $u(c_2)=0.25$
> - 「50%で100万円・50%で $c_1$ 円」で迷う確実額 $c_3$ $\to$ $u(c_3)=0.75$

これで効用が $0,\ 0.25,\ 0.5,\ 0.75,\ 1$ の**5点**が手に入る．横軸＝金額，縦軸＝効用でプロットする．

### グラフを描いて「自分のタイプ」を読む

```{important} 読み取り方
- 点を結んだ線が**上に凸**（$c_1$ が50万円より小さい，例えば30万円）$\to$ <span style="color:red">リスク回避的</span>．
- ほぼ**直線**（$c_1 \approx 50$ 万円）$\to$ <span style="color:red">リスク中立的</span>．
- **下に凸**（$c_1 > 50$ 万円）$\to$ <span style="color:red">リスク愛好的</span>．
```

◆ 隣の人と $c_1$ を見せ合う．**同じ問いでも答えが違う** ＝ 効用関数は人それぞれ．これが「個人差を数理で捉える」ということである．

```{note} 板書・電卓だけで完結（PC不要）
配布する記入シート例：

| 効用の値 | 対応する金額（あなたの確実性等価） |
| --- | --- |
| 0 | 0円 |
| 0.25 | （　　　）円 |
| 0.5 | （　　　）円 |
| 0.75 | （　　　）円 |
| 1 | 100万円 |

方眼黒板に5点を打って手で結ぶだけで「自分の効用曲線」が完成する．
```

:::{seealso} （PCがあれば）自分の5点に数式モデルを当てはめる
集めた5点に，べき乗の効用関数 $u(x) = (x/100)^{\alpha}$（$x$ は万円）を**フィッティング**すると，たった1つの数 $\alpha$（リスク態度パラメタ）で効用関数を要約できる．

```python
import numpy as np
from scipy.optimize import curve_fit

# 例：ある受講者が答えた確実性等価（万円）。u=0.25,0.5,0.75 に対応
money = np.array([0, 30, 50, 72, 100])   # 自分の確実性等価に置き換える
util  = np.array([0, 0.25, 0.5, 0.75, 1.0])

def u(x, alpha):
    return (x / 100.0) ** alpha

alpha, _ = curve_fit(u, money, util, p0=[1.0], bounds=(0.05, 5))
print(f"あなたの alpha = {alpha[0]:.2f}")
print("alpha<1: リスク回避 / =1: 中立 / >1: リスク愛好")
```

クラス全員の $\alpha$ を集めてヒストグラムにすれば，その場で「この教室の意思決定データ」ができる．これは実験経済学・データサイエンスの第一歩である．大学の[第2回](/contents/2_expected_utility)では，これをさらに本格化した「CRRA効用関数による最尤推定」も扱う．
:::

## もう一歩先：「夢」を数理に入れる（プロスペクト理論・発展）

効用関数でも，まだ説明しきれないことがある．**当選確率1千万分の1**という極小の確率を，人は本当に「ほぼ0」として扱っているだろうか．

```{important} 確率の感じ方は確率そのものとずれている
カーネマンとトベルスキーの**プロスペクト理論**（1979，のちにノーベル経済学賞）によると，人は確率 $p$ をそのまま使わず，**主観的な重み $w(p)$** に変換して評価する．

- **小さい確率は過大評価**：1千万分の1を，心の中では「もっとありそう」と感じる．
- **大きい確率は過小評価**する．

宝くじが売れる大きな理由のひとつがこれである．さらに**買った瞬間から抽選日まで「もし当たったら」と夢を見られる**．その夢の時間も含めて300円を払っている，とも解釈できる．
```

```{note} つまり，宝くじを買う人は「不合理」なのか
- 期待値だけ見れば損．
- だが「効用関数の形」＋「確率の感じ方」＋「夢という体験」まで入れると，**宝くじを買うことは，その人にとって筋の通った（合理的な）選択**になりうる．

数理は「人を不合理だと裁く道具」ではなく，**「なぜそう行動するのかを理解する道具」**である．ここが現代の意思決定研究の最前線である．
```

## 付録B：想定問答（Q&A）

```{note} Q. 期待値が損でも，当たる人は実際にいますよね？
A. その通り．期待値は「全員を平均した」値で，個人の結果は当たり外れに大きくばらつく．「ほとんどの人が0円，ごく少数が巨額」という偏った分布になる．期待値だけでなく**分布の形**を見るのが大事，というのも今日の裏テーマである．
```

```{note} Q. では宝くじは買わないほうがいい？
A. 数学は「正解」を押し付けない．**自分の効用関数と，夢という体験をどう評価するか**で答えは変わる．「無理のない範囲で夢を買う」のは，その人にとって合理的でありうる．大事なのは**自分の判断を説明できること**である．
```

```{note} Q. 効用関数は本当に測れるの？
A. 今日の確実性等価法は実際の研究手法の簡易版である．より精密には多数の質問・実験で推定する．測り方そのものが研究テーマであり，心理学・経済学・統計学が交わる面白い領域である．
```
