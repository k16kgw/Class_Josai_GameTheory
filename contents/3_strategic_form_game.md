# 第3回：ナッシュ均衡

## ナッシュ均衡の定義

**ナッシュ均衡（Nash Equilibrium）** は，ゲーム理論における最も重要な解概念である．
1950年にジョン・ナッシュが提唱し，1994年にノーベル経済学賞を受賞した．

```{admonition} 定義：ナッシュ均衡
戦略の組 $s^* = (s_1^*, s_2^*, \ldots, s_n^*) \in S$ が**ナッシュ均衡**であるとは，
すべてのプレイヤー $i \in N$ と，すべての戦略 $s_i \in S_i$ に対して：

$$u_i(s_i^*, s_{-i}^*) \geq u_i(s_i, s_{-i}^*)$$

つまり，**他の全プレイヤーが均衡戦略をとるとき，どのプレイヤーも戦略を変える誘因がない**状態．
```

## 直感的な解釈

- 「誰も後悔しない」戦略の組み合わせ
- 各プレイヤーが相手の戦略を正しく予測し，それに対して最善を尽くしている状態
- 自己実現的な予測（Self-Fulfilling Prophecy）

## ナッシュ均衡の求め方

### 方法1：最適反応関数（Best Response Function）

```{admonition} 定義：最適反応
プレイヤー $i$ の**最適反応関数** $BR_i(s_{-i})$ は，相手の戦略 $s_{-i}$ に対して $i$ の利得を最大化する戦略の集合：

$$BR_i(s_{-i}) = \arg\max_{s_i \in S_i} u_i(s_i, s_{-i})$$

ナッシュ均衡 $s^*$ は，すべての $i$ について $s_i^* \in BR_i(s_{-i}^*)$ を満たす．
```

### 方法2：利得行列での下線法

各列でプレイヤー1の最大利得に下線，各行でプレイヤー2の最大利得に下線を引く．
**両方に下線が引かれているセル**がナッシュ均衡候補．

## 具体例1：囚人のジレンマ

|  | C | D |
|--|---|---|
| **C** | (3, 3) | (0, 5) |
| **D** | (5, 0) | (1, 1) |

下線法を適用：

|  | C | D |
|--|---|---|
| **C** | (3, **3**) | (0, **5**) |
| **D** | (**5**, 0) | (**1**, **1**) |

ナッシュ均衡：**(D, D) = (自白, 自白)**

```{note}
(C,C) = (3,3) の方が双方にとって良いが，ナッシュ均衡ではない．
どちらも相手がCをとるなら自分はDに変えた方が利得が上がるため．
```

## 具体例2：男女の闘い（Battle of the Sexes）

夫はサッカー（S），妻はオペラ（O）が好きだが，一緒に行動したい．

|  | S | O |
|--|---|---|
| **S** | (**2**, 1) | (0, 0) |
| **O** | (0, 0) | (1, **2**) |

下線法：

|  | S | O |
|--|---|---|
| **S** | (**2**, **1**) | (0, 0) |
| **O** | (0, 0) | (**1**, **2**) |

ナッシュ均衡：**(S, S)** と **(O, O)** の2つが存在．

```{tip}
ナッシュ均衡は複数存在することがある．どちらの均衡が実現するかは，
コーディネーション問題（Coordination Problem）として別途分析が必要．
```

## 具体例3：均衡が存在しない場合？

じゃんけん：$S_1 = S_2 = \{R, S, P\}$

|  | R | S | P |
|--|---|---|---|
| **R** | (0,0) | (1,-1) | (-1,1) |
| **S** | (-1,1) | (0,0) | (1,-1) |
| **P** | (1,-1) | (-1,1) | (0,0) |

純粋戦略ナッシュ均衡は存在しない（どの組み合わせも相手が変えたい）．

```{note}
ナッシュ（1950）は，有限ゲームには**混合戦略**を含めると必ず均衡が存在することを証明した．
混合戦略については第4回で学ぶ．
```

## ナッシュ均衡の性質

| 性質 | 内容 |
|------|------|
| 存在性 | 有限ゲームでは混合戦略ナッシュ均衡が必ず存在する |
| 一意性 | 一般に一意ではない（複数存在しうる） |
| 効率性 | パレート最適でないこともある（囚人のジレンマ） |

## まとめ

- ナッシュ均衡は「誰も変更の誘因がない」戦略の組み合わせ
- 最適反応関数の交点，または下線法で求められる
- 複数存在することも，存在しないこともある（純粋戦略の場合）
- 次回はじゃんけんのような純粋戦略均衡のないゲームを扱う**混合戦略**を学ぶ
